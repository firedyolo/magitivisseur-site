Vous êtes sur PC, voici donc le guide d'installation de la version PC (Windows, sauf si vous êtes un petit bidouilleur)

Premièrement, rendez-vous sur le Github du projet.

Ensuite faîtes un clique droit sur votre dossier -> extraire. Si :

  • ■ Vous avez déjà NodeJs sur votre PC, ouvrez le fichier "start.bat - Raccourci" (si celui-ci ne fonctionne pas, allez dans le dossier "Convertisseur" et ouvrez le fichier "start.bat")
  • ■ Vous n'avez pas NodeJs sur votre PC, lisez le guide d'installation de NodeJs fourni dans l'archive précédement téléchargée.

Vous êtes sur téléphone, voici donc le guide d'installation de la version Android. Rendez-vous sur le Github pour voir le code !

  • Si vous aviez déjà installé auparavant une version du Magitivisseur, merci de la désinstaller (le discord permet de se tenir informé des mises-à-jour)
  • Téléchargez ce .apk depuis ce bouton : (lien Mega)
  • Exécutez l'application sur votre téléphone
  • Autorisez l'application à accéder à vos fichiers. Sans cette autorisation, celle-ci ne peut enregistrer vos musiques
